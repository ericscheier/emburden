# emburden: Energy Burden Analysis Using Net Energy Return Methodology

Provides tools for calculating and analyzing household energy burden
using the Net Energy Return (Nh) aggregation methodology. Functions
support weighted statistical calculations across geographic and
demographic cohorts, with utilities for formatting results into
publication-ready tables. Based on methods from "Net energy metrics
reveal striking disparities across United States household energy
burdens".
